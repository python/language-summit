Python Language Summit
======================

Issue Tracker and Discussions: https://github.com/python/language-summit

For questions and suggestions about the language summit, please open an issue or
a discussion item in the `language-summit
<https://github.com/python/language-summit>`_ repo.

Python Language Summit 2021 is scheduled over two days: **May 11 (Tuesday) and
May 12 (Wednesday)**, it will be hosted virtually over Zoom.

More details to come!

**Help improve this documentation** by opening an issue or creating a pull
